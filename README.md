# About
Viewer based on [Godot](https://godotengine.org/) and the [Leia glTF Exporter](https://e-verse.com/leia-plugin/). 

# Getting Started

### Run on your PC

- Install Godot
- Load Scene node_3d.tscn from BIM Viewer

Run a local server to test your model
